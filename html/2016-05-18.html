  public $dsParamFILTERS = array(
    '141' => '{$url-community:no}'
  );

  public function execute(array &$param_pool = null)
  {
    $result = new XMLElement($this->dsParamROOTELEMENT);
    if ($this->dsParamFilters[141] == ‘no’) return; 
  }
I took the lazy way showing you a ‘return’ but you should really output some basic XML saying that the execution was stopped. otherwise continue with your normal datasource
